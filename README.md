# LeadPilot for Real Estate

A lightweight browser app for real estate agents to qualify buyer leads and generate the next best SMS response.

## What it does

- Captures buyer lead context
- Generates concise SMS replies under 320 characters
- Asks one qualification question at a time
- Flags hot buyer intent inside 90 days
- Routes legal, pricing, contract, commission, offer, mortgage, and financial questions to human follow-up
- Keeps agent setup tucked away to reduce day-to-day friction

## Run locally

Open `index.html` in a browser.
